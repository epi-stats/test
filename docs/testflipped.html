Basics of using R

R as calculator

R is as a powerful calculator. In addition to the simple arithmetic functions +, -, /, * R computes

  • Power: x^y
  • Square root: sqrt(x)
  • Cubic root: x^(1/3)
  • Logarithm: log(x), log10(x), log(x, base=y)
  • Exponentiation: exp(x)
  • Modulus: x %% y
  • Trigonometric functions …

To get a first R-exposure, compute the results of the following mathematical expressions:

2 + 3 * 4
2^1023
sqrt(4)
4^0.5
-2^2
2^1^3

The last 2 lines in the example above gave surprising results. Do you have any idea why R calculates -22 = -4?

Logical operators

R has the usual typical operators

  • x > y Returns TRUE if x is larger than y (otherwise FALSE)
  • x >= y Returns TRUE if x is larger than or exactly equal to y
  • x < y Returns TRUE if x is smaller than y
  • x <= y Returns TRUE if x is smaller than or exactly equal to y
  • x == y Returns TRUE if x exactly equals y
  • x != y Returns TRUE if x differs from y
  • x & y Returns TRUE is X is TRUE and y is TRUE
  • x | y Returns TRUE is X is TRUE or y is TRUE

Let’s see how it works:

5 > 1
5 < 1
5 <= 5
1 - 0.6 == 0.4
1 - 0.9 == 0.1

Vectors and vector operations

Assign expression to objects

After you enter an expression in the R terminal the result will be displayed immediately. However, in most cases you would like to have the result stored in an object that you can access it later on. To assign the result to an object use the ‘<’ followed by a ‘-’. You can choose any name for an object (R is case sensitive). Enter the object name at a later stage R will display the content of the object. If you would like to assign and display the result, enclose the expression in round brackets, e.g. (five <- 2 + 3) Now, try the code below line by line.

a <- 2 
b <- a + a
a
[1] 2
b
[1] 4

Vectors

A vector is a sequence of items of the same type (often numbers). A vector can be created using the combine function c, e.g. c(item1, item2, item3). All items have to be of the same type or will be transformed into the same type, e.g. if one of the items contains non-numerical characters all items will be converted into text. To generate sequences you have several options. The expression 1:10 is synonym for the integer numbers from 1 to 10. An alternative way is to use the function rep(number, times) for instance rep(3, 4) would create the vector 3, 3, 3, 3. The function seq generates a logical sequence either by seq(start, end, increment).

Try to generate the following vectors:

x <- c(3, 5, 7)
y <- 3:12
z <- seq(3, 9, 2)
x
y
z
